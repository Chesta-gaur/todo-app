# todo-app
A clean and responsive To-Do List Web App built with HTML, CSS, and JavaScript that allows users to add, edit, delete, and mark tasks as complete. The app stores data using Local Storage, ensuring tasks remain saved even after refreshing the browser.

---

A simple and efficient To-Do List Application built to help users manage their daily tasks with ease and productivity. This project focuses on building a clean, responsive, and user-friendly interface using core frontend technologies.
